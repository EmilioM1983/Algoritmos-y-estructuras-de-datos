Propósito: Sumar dos números representados por listas enlazadas simples, donde cada nodo contiene un dígito en orden inverso (el primer nodo es el dígito menos significativo).

Algoritmo: Recorre ambas listas simultáneamente, sumando dígitos y manejando el “carry” (acarreo) si la suma supera 9. Crea una nueva lista con los dígitos resultantes.

Ejemplo: 1→2→4 + 5→2→8 → 6→4→2→1
(6421 + 825 = 7246)

Complejidad:
- Temporal: O(n)
- Espacial: O(n))
